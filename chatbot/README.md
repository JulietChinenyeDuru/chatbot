# ChatBot

A simple, interactive chatbot with a clean interface.

## Features

- Real-time chat interface
- Responds to greetings, questions, and commands
- Shows current time and date
- Tells programming jokes
- Smooth animations
- Responsive design

## Usage

Simply open `index.html` in your browser and start chatting!

## Try asking:

- "Hi" or "Hello"
- "What's your name?"
- "What time is it?"
- "Tell me a joke"
- "Help"

## Extending the Bot

To add more responses, edit the `generateResponse()` function in `script.js`.
